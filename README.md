Ce dépôt présente un exercice de compréhension orale simple du Vietnamien. 
On écoute d'abord un mot ( ou une expression ) , puis choisis une réponse dans un QCM ( 3 choix possibles ).
En cliquant sur un autre bouton, on obtient alors une correction texte + Audio. 
L'objectif est de rendre facile l'insertion de contenu dans cette application javascript, pour l'instant côté Navigateur
mais bientôt via NodeJS. 
Le fonctionnement de l'application fonctionne pour l'instant via JS.
